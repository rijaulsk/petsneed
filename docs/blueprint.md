# **App Name**: Pawsitive Needs

## Core Features:

- Informational Content Display: Display comprehensive information about pet grooming, clinic services, and available products.
- Contact and Location Integration: Provide easy access to contact information, store hours, and location details with map integration.
- Customer Reviews Display: Showcase customer testimonials to build trust and credibility.
- Pet Breed Identification Tool: AI-powered tool to identify pet breeds from user-uploaded images, providing care tips.

## Style Guidelines:

- Primary color: Coral Orange (#FF7A5C) for a friendly, warm, and inviting feel.
- Background color: Warm Cream (#FFF8F3), providing a soft, clean backdrop to ensure readability and comfort.
- Accent color: Mint Green (#A8E6CF) to highlight key sections, promotions, or important pieces of information.
- Headline font: 'Fredoka' (sans-serif) for a playful and modern touch, especially effective in headings. Note: currently only Google Fonts are supported.
- Body font: 'Quicksand' (sans-serif) offers a clean, readable style suitable for all body content. Note: currently only Google Fonts are supported.
- Use playful, rounded icons throughout the app, specifically related to pet care (paw prints, food bowls, stethoscopes).
- Use a clear and intuitive layout to easily navigate to different sections.
